# iOS native receiver (optional)

The **web app** in this repo cannot drive real Taptic / Core Haptics on iPhone. A **thin native iOS app** (WebView or native UI + your pairing protocol) is the path to physical haptics for beta testers.

This folder holds **distribution docs** for a small closed beta. The actual Xcode project is created locally (or added here later) with your team’s bundle ID and signing assets.

- **[Ad Hoc distribution (up to 4 devices)](./ADHOC-DISTRIBUTION.md)** — register UDIDs, export an IPA, install without the public App Store.
